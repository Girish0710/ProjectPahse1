Welcome to File manager Application
How to use Application
1)User can select option provided to dig in
2)First option -Show files: User can view all the files present in the file
3)Second  option have feature to add,delete, and search the files
4)Add files : user can add ".txt" file
5)Delete file : User can delete files present in the folder
6)Search file : User can view the application searched in position showed as index
7)Exit option to drive back to Main menu -Option 4
8)Main Exit-option 3: And Thanks message from developer
